Introdução

HTML (HyperText Markup Language) é a linguagem utilizada para criar a estrutura de páginas web.
Com ele, definimos títulos, parágrafos, imagens, links e outros elementos que compõem um site.

Estrutura Básica

Todo documento HTML possui uma estrutura padrão:
<!DOCTYPE html> → define o tipo do documento
<html> → raiz da página
<head> → configurações (título, meta, etc.)
<body> → conteúdo visível ao usuário

Principais Tags

Algumas das tags mais utilizadas no HTML são:
<h1> até <h6> → títulos
<p> → parágrafos
<a> → links
<img> → imagens
<div> → organização de conteúdo

Listas

O HTML permite criar listas organizadas:
<ul> → lista não ordenada
<ol> → lista ordenada
<li> → item da lista

Formulários

Formulários são usados para coletar dados do usuário:
<input> → campos de entrada
<textarea> → texto longo
<select> → lista de opções
<button> → botão

Conclusão

O HTML é a base do desenvolvimento web, sendo responsável pela estrutura das páginas.
Ele trabalha junto com CSS e JavaScript para criar sites completos e interativos.
